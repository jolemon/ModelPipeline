模型报告生成器 — 项目框架概览

基于 readme.md 规范，将评分卡模型 + 打分结果 → Excel 模型报告

📥 输入

1. 模型文件 (.pkl)
训练好的评分卡对象
包含: Binner + Scorecard 实例
2. 打分结果 (CSV/DataFrame)
part_id, cert_no, loan_date,
mob6_30, data_flag, pred_score,
scorecard_score, feat1, feat2...

⚙️ 处理引擎 (ReportGenerator)

分区分布统计
变量 IV/KS/PSI 分析
单变量 WOE 分箱
评分卡详情
模型效果评估
分箱表现分析

📤 输出

Excel 报告 (多个 sheet)
Sheet 1模型设计样本分区分布 + 建模分
Sheet 2变量分析IV/KS/PSI 表格 + Top10 单变量 WOE
Sheet 3模型表现评分卡详情 + 建模效果 + 回溯 + 分箱表现

🔗 依赖关系

scorecard_jsb.py (Binner/Scorecard) → 仅做参考，了解评分卡内部结构（分箱、WOE、系数）
报告生成器是新的独立模块，消费已训练好的评分卡对象
