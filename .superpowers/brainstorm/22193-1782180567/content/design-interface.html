ScorecardProtocol 接口定义

报告生成器只需要这些信息，来自 scorecard 对象

class ScorecardProtocol(Protocol):
  # 变量分箱信息
  def get_var_names(self) → list[str]: ...
  def get_bins(self, var: str) → pd.Series: ...
  def get_woe_table(self, var: str) → pd.DataFrame: ...
  def get_iv_table(self) → pd.Series: ...
  def get_ks_table(self) → pd.Series: ...
  # 模型信息
  def get_model_summary(self) → pd.DataFrame: ... # Parameter/Estimate/Std-Error/Wald-Chi2/P-value/VIF
  def get_scorecard(self) → pd.DataFrame: ... # name/left/right/score
  def get_missing_dict(self) → dict: ...
  def get_dropped_vars(self) → list[str]: ...

错误处理策略

缺少必需列 抛 DataValidationError，列出缺失的列名
data_flag 缺少 train/test/oot 抛 DataValidationError，提示哪些必需的 data_flag 值缺失
target 不是 {0,1} 抛 DataValidationError
变量元数据文件缺失 Warning，相关列填空字符串，不中断报告生成
变量分箱信息缺失 跳过该变量的 WOE 表，变量分析表中标注 "未分箱"
压测集缺失 可接受，3.3 中 oos 行不生成
